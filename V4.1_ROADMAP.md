# FreeLang v4.1 Roadmap

## 목표
v4.0 (v1.0-stable)의 완성도를 높이고, 실용적인 도구 추가

**버전**: 4.1.0
**출시 목표**: 2026년 4월
**테스트 목표**: 260+ (현재 252)

---

## 📋 v4.1 구현 항목

### Phase 1: 핵심 버그 수정 (1주)
- [ ] Pattern matching 테스트 12개 실패 원인 파악 & 수정
- [ ] JSON stringify 추가 케이스
- [ ] 파일 I/O 에러 처리 개선

### Phase 2: 기본 도구 추가 (1주)
- [ ] **REPL** (대화형 쉘) ← 가장 요청 많음
- [ ] `--version` 플래그
- [ ] `--help` 개선
- [ ] `--dump-ast` 옵션

### Phase 3: 표준 라이브러리 최소 버전 (2주)
```
freelang-stdlib/
├── collections/
│   ├── HashMap<K,V>
│   └── HashSet<T>
├── string/
│   ├── split (개선)
│   ├── join
│   ├── startsWith
│   └── endsWith
└── math/
    └── random (0~1)
```

### Phase 4: 예제 & 문서 (1주)
- [ ] 5개 실전 예제
  - `hello.fl` - 기본
  - `fibonacci.fl` - 재귀
  - `http-client.fl` - 네트워크
  - `file-processor.fl` - I/O
  - `web-scraper.fl` - async+HTTP

## 🔧 상세 구현

### 1️⃣ REPL 구현

**파일**: `src/repl.ts`

```typescript
class REPL {
  // 대화형 쉘
  // 1. 각 줄을 컴파일 & 실행
  // 2. 이전 정의 기억 (변수, 함수)
  // 3. 다중 라인 입력 지원
  // 4. 히스토리 & 자동완성
}
```

**사용**:
```bash
freelang --repl
> var x = 10
> println(x)
10
> fn add(a, b) { a + b }
> add(3, 4)
7
>
```
